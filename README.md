This repository is made for openSUSE.

# セットアップ

github cli をインストール  
`https://github.com/cli/cli/blob/trunk/docs/install_linux.md`

`dotfiles`をクローン  
`gh repo clone kapiyva/dotfiles ~/dotfiles`

インストールスクリプトを実行  
`sudo chmod +x ~/dotfiles/install.sh && ~/dotfiles/install.sh`
